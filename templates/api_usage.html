YAGRA API

Like the api from Gravatar, YAGRA provides api to allow you to pass the md5 digest of the user email to get the avatar of the user.

All you need to do is the send a GET request to
http://cenhao.chinacloudapp.cn/yagra/api
with key = md5(user lowercase email)

Example:

http://cenhao.chinacloudapp.cn/yagra/api?key=3f25bd7f9187e7f44d4bd89eba6b74d9

is to get the avatar of user cenhao123@gmail.com, where md5('cenhao123@gmail.com'.lower()) == '3f25bd7f9187e7f44d4bd89eba6b74d9'
